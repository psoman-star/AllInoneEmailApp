## ✨ Key Features

### 🔧 Search Settings
| Setting | Description |
|---|---|
| **Engine Country** | Target search results from any country worldwide |
| **Language** | Filter results by language (40+ supported) |
| **MaxNum** | Set maximum extraction limit (up to 500,000+) |
| **Threads** | Multi-threaded extraction — more threads = faster speed |
| **Delay** | Control request interval to avoid rate limits |
| **Proxy** | HTTP proxy support with rotation for unblocked extraction |

### 📬 Email Extraction
- Extract **Person emails** (@gmail, @yahoo, @hotmail, etc.)
- Extract **Business emails** (company domains)
- Fully editable email domain list
- Filter by domain, country, keyword, and extension

### 🌐 Social Network Support
Extract contacts directly from:

| Platform | Platform | Platform |
|---|---|---|
| ✅ Facebook | ✅ Twitter / X | ✅ LinkedIn |
| ✅ Instagram | ✅ TikTok | ✅ YouTube |
| ✅ Pinterest | ✅ Tumblr | ✅ Dribbble |
| ✅ VK | ✅ Discord | ✅ OK.ru |
| ✅ Line | ✅ Snapchat | ✅ AliExpress |
| ✅ SoundCloud | ✅ Any Custom Site | ✅ Google Maps |

### 🗺️ Google Maps Data Extractor
Extract rich business data from any city or keyword:

- 🏢 Business Name
- ⭐ Rating & Reviews
- 📍 Address, Longitude, Latitude
- 📞 Phone Number
- 📧 Email Address
- 🌐 Website URL
- 🕐 Business Hours
- 🖼️ Images

### 🌍 Any Website Support
Add any custom domain (e.g. `soundcloud.com`, `aliexpress.com`) and extract contact data from it directly using keyword-driven search.

### 🔑 Keyword Engine
- Supports **bulk keywords** (one per line)
- Run continuously without stopping by adding more keywords
- Combine country codes + job titles for targeted leads (e.g. `+91 ceo`, `+44 manager`)

### 🛡️ Proxy Manager
- Import proxies from **TXT** or **CSV** files
- Built-in proxy **tester** — validates all proxies before use
- Remove failed proxies in one click
- Rotate proxies automatically during extraction to avoid IP blocks

### 💾 Export Options
Export your results to:
`Excel (.xlsx)` · `CSV` · `JSON` · `HTML` · `SQL`
